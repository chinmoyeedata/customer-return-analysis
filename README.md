# Customer Return Analysis
High product return rates in e-commerce can reduce profitability and customer satisfaction.  
This project aims to analyze return patterns and identify key factors contributing to returns.

## Dataset
- E-commerce sales and return dataset  
- Includes product category, region, discount, and return status

## Tools Used
- Python (Pandas, Data Analysis)
- Tableau (Dashboard Visualization)

## Key Insights
- Office Supplies category shows the highest return volume  
- Higher discounts are associated with increased returns  
- Returned products generate lower profit for the business

## Business Recommendations
- Improve product quality in high-return categories such as Office Supplies  
- Re-evaluate discount strategies to reduce unnecessary returns  
- Strengthen logistics and delivery processes in regions with high return rates
